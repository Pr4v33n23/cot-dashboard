# Intelligence Page — Design Spec
**Date:** 2026-05-18
**Status:** Approved

---

## 1. What this builds

A single `/intelligence` route — a combined daily digest + per-market deep-dive powered by DeepSeek-V4-Pro. Left panel shows the global cross-market digest (macro narrative, sector signals, top watch markets, macro + micro news feed). Right panel loads a full per-market synthesis when the user clicks any market.

Replaces the `SynthesisDrawer` design from the previous spec — synthesis is now a right-panel on this dedicated route, not a drawer overlay on the market detail page. The market detail page's IntelStrip "Confluence" cell deep-links to `/intelligence?sym=SYMBOL`.

---

## 2. Layout

```
/intelligence
├── Left panel (340px fixed, scrollable)
│   ├── Page header (title, DeepSeek badge, date)
│   ├── Macro narrative card
│   ├── Sector signals card (6 rows: energy/metals/grains/FX/indices/rates)
│   ├── Top watch markets (clickable rows, active = highlighted)
│   └── Macro + micro news feed
└── Right panel (fills remainder, scrollable)
    ├── Market header (symbol, name, sector, market_type badge, zone badges)
    ├── Confluence strip (score, progress bar, factor chips)
    ├── COT breakdown (4 mini-cards: commercials, large specs, small specs, divergence)
    ├── DeepSeek analysis paragraph
    ├── Dual news feed (macro column | micro column, each scored)
    └── Watch item
```

Right panel default state (no market selected): empty state with prompt "Select a market from the watch list".

## 3. New API endpoints

### `GET /intelligence/digest`

Weekly cross-market digest. Generated by DeepSeek-V4-Pro after CFTC release. Cached in `research/data/cache/intelligence_digest.json`, rebuilt on Friday cron.

**Input to DeepSeek** (structured, not freeform):
```python
{
  "date": "2026-05-18",
  "top_markets": [  # top 10 by confluence_score from synthesis.parquet
    {"symbol": "CL", "confluence_score": 0.84, "key_factors": [...], "regime": "accumulation"},
    ...
  ],
  "sector_summary": {  # per sector: avg confluence, avg cot_index, dominant signal
    "energy": {"avg_confluence": 0.79, "avg_cot_index": 87.2, "n_markets": 5},
    ...
  },
  "macro_news": [  # FOMC, OPEC, WASDE, EIA from past 7 days, avg sentiment
    {"source": "OPEC", "title": "...", "sentiment_score": 0.82},
    ...
  ],
  "divergence_count": 8,  # markets with active comm_spec_divergence > 0
  "regime_counts": {"accumulation": 12, "trending": 8, "distribution": 5, "ranging": 22}
}
```

**Output** (stored in `intelligence_digest.json`):
```json
{
  "generated_at": "2026-05-18T16:00:00Z",
  "macro_narrative": "...",
  "sector_signals": [
    {"sector": "energy", "summary": "...", "signal": "bullish"},
    ...
  ],
  "watch_markets": [
    {"symbol": "CL", "confluence_score": 0.84, "reason": "..."},
    ...
  ]
}
```

**Response model (`DigestResponse`):**
```python
class SectorSignal(BaseModel):
    sector: str
    summary: str
    signal: Literal["bullish", "bearish", "neutral"]

class WatchMarket(BaseModel):
    symbol: str
    name: str
    sector: str
    confluence_score: float
    reason: str

class DigestResponse(BaseModel):
    generated_at: datetime
    macro_narrative: str
    sector_signals: list[SectorSignal]
    watch_markets: list[WatchMarket]
```

### Existing `GET /synthesis/{symbol}` — already built

Used by the right panel when user clicks a market. Returns `SynthesisResponse` (already in schemas).

### Existing `GET /news/{symbol}` — already built

Used by the right panel dual news feed. Add `?macro_only=true` query param to filter to scheduled events (FOMC, OPEC, WASDE, EIA).

Builds the digest payload and calls DeepSeek-V4-Pro:

```python
def build_digest_payload(

def generate_digest(payload: dict) -> dict: ...  # calls _ai.chat()

def load_or_generate_digest(
    annotated, news_df, synthesis, cache_dir: Path

```

**System prompt for digest:**
```
You are a quantitative market analyst producing a weekly intelligence digest.
Given structured COT, regime, and news data across 47+ futures markets, produce:
- macro_narrative: 3-4 sentences covering the dominant cross-market theme this week
- sector_signals: one-line summary + signal ("bullish"/"bearish"/"neutral") per sector
- watch_markets: top 5 markets with a one-line reason each

All statements must be derived from the data provided. No invented facts.
No trading advice. No predictions. Factual context only.
Return JSON only.
```

## 5. Frontend

### New route: `src/routes/intelligence/+page.svelte`

**State:**
```typescript
let digest = $state<DigestResponse | null>(null);
let selectedSym = $state<string | null>(null);  // from URL ?sym= or click
let synthesis = $state<SynthesisResponse | null>(null);
let marketDetail = $state<MarketDetail | null>(null);
let newsForSym = $state<NewsItem[]>([]);
```

**On mount:** load digest. If `?sym=SYMBOL` in URL, auto-select that market and load its synthesis.

**On market select:** `api.synthesis(sym)` + `api.newsForSymbol(sym, { limit: 20 })` in parallel.

**Left panel sections:**
1. Macro narrative — rendered text from `digest.macro_narrative`
2. Sector signals — 6 rows (or however many returned), signal → colour class
3. Watch markets — `digest.watch_markets`, clicking sets `selectedSym`
4. News feed — pulled from `/news` (all markets, last 7 days, macro sources only), shown with `nm-macro`/`nm-micro` type tags and `sentiment_score`

**Right panel sections (when `selectedSym` is set):**
1. Header — symbol, name, sector, `market_type` badge, active zone badges
2. Confluence strip — `synthesis.confluence_score`, `synthesis.key_factors` as chips
3. COT breakdown — last bar from `marketDetail.bars` — `net_commercials`, `mm_long/short`, `nr_long/short`, `comm_spec_divergence`/`am_lf_divergence`
4. Analysis — `synthesis.summary` paragraph
5. Dual news feed — `newsForSym` split into `macro` (source_category = "macro" | "agency") and `micro` (source_category = "market") columns, scored with `sentiment_score`
6. Watch item — `synthesis.watch`

**Right panel empty state:** `<EmptyState title="Select a market" body="Click any market in the watch list to load its full analysis." />`

## 7. Data flow

```
Friday cron
  → build_bundle() [COT + prices + TFF + zones + regime + news sentiment]
  → synthesize_all() [DeepSeek Pro per market → synthesis.parquet]
  → generate_digest() [DeepSeek Pro cross-market → intelligence_digest.json]
  → /refresh

User opens /intelligence
  → GET /intelligence/digest  [reads intelligence_digest.json]
  → left panel renders: narrative + sectors + watch markets

User clicks CL
  → GET /synthesis/CL         [reads synthesis.parquet]
  → GET /market/CL            [last 90d bars for COT breakdown]
  → GET /news/CL?limit=20     [scored headlines]
  → right panel renders: COT + analysis + dual news + watch
```

None — `_ai.py` and `market_synthesis.py` already handle DeepSeek calls. This is a new orchestration module on top of existing infrastructure.
